**Sistema de Voluntariado**
Sistema de consola para administrar personas de una organización solidaria, diferenciando entre Voluntarios y Coordinadores, con información y comportamiento específico para cada rol.

**Descripción**
Una organización solidaria necesita un sistema para gestionar a sus integrantes. El sistema contempla dos tipos de personas:

Voluntarios — Integrantes que realizan tareas y acumulan horas de trabajo.
Coordinadores — Responsables de áreas específicas y de gestionar equipos de personas.

Cada tipo tiene información y comportamiento propio, pero comparten una base común que permite tratarlos de forma unificada dentro del sistema.

**Funcionalidades**

*Registrar voluntarios y coordinadores en el sistema
*Mostrar la información general de cada integrante
*Visualizar las horas trabajadas de cada voluntario
*Visualizar el área asignada y la cantidad de personas a cargo de cada coordinador
*Listar todos los integrantes de la organización de forma polimórfica
*Diferenciar el comportamiento de cada tipo al mostrar su información
